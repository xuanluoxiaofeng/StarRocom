# 更新日志

## v1.0.0 (2026-04-07)

### 🎉 首次发布

#### ✨ 新功能

- **图鉴浏览**
  - 完整的精灵图鉴展示
  - 6列网格布局
  - 支持分页浏览
  - 精灵卡片展示（图片、名称、类型、形态）

- **精灵详情**
  - 点击精灵卡片查看详情
  - 模态框展示，背景模糊效果
  - 基础属性展示（HP、攻击、防御等）
  - 进化链可视化（支持分支进化）
  - 技能列表（分页显示，每页6个）
  - 孵化信息展示

- **精灵查蛋**
  - 根据尺寸和体重筛选精灵
  - 按蛋组分类展示
  - 无法孵蛋组筛选
  - ✨异色精灵筛选
  - 去重显示（相同名称精灵只显示第一个）
  - 与图鉴一致的UI风格

- **界面优化**
  - 仿宝可梦图鉴设计
  - 响应式卡片布局
  - 平滑动画效果
  - 悬停交互反馈
  - 精美的类型徽章

#### 🔧 技术特性

- **Electron框架**
  - 桌面应用封装
  - GPU加速控制
  - 代理支持（用于网络请求）

- **数据管理**
  - JSON数据集中加载
  - 进化数据独立加载
  - 精灵详情按需加载
  - 异色精灵检测

- **性能优化**
  - 图片懒加载
  - 预加载下一页数据
  - GPU加速开关
  - 禁用GPU错误修复

## 开发者说明

### 构建应用

```bash
# 安装依赖
npm install

# 开发模式
npm start

# 打包Windows版本
npm run build:win

# 打包所有平台
npm run build
```

打包后的安装包位于 `dist/` 目录。

### 代理设置

如需使用代理，请在命令行中设置：
```bash
set http_proxy=http://192.168.155.153:7890
set https_proxy=http://192.168.155.153:7890
```
