# 洛克王国精灵图鉴 - 打包说明

## 构建步骤

### 1. 安装依赖
```bash
npm install
```

### 2. 打包应用
```bash
npm run build:win
```
或
```bash
npm run build
```

打包完成后，安装包会在 `dist` 目录中。

## 输出文件

打包后会生成以下文件：
- `dist/洛克王国精灵图鉴 Setup x64.exe` - Windows 安装程序
